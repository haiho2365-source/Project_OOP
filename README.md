# XÂY DỰNG CHƯƠNG TRÌNH QUẢN LÝ BẢN TIN CỦA ĐÀI TRUYỀN HÌNH

##  Tổng quan (Introduction)
Chào mừng bạn đến với dự án **Chương trình Quản lý Bản tin Đài Truyền hình**.

**Nội dung của đề tài:**
Đây là phần mềm được thiết kế nhằm hỗ trợ quy trình sản xuất và phát sóng tin tức tại các đài truyền hình. Hệ thống giúp tối ưu hóa việc quản lý luồng công việc từ khâu soạn thảo, biên tập, kiểm duyệt cho đến khi bản tin được lên lịch phát sóng.

Các chức năng chính của chương trình bao gồm:
* **Quản lý tin tức:** Soạn thảo, chỉnh sửa, lưu trữ và phân loại các bản tin theo chủ đề (Chính trị, Xã hội, Thể thao, Thời tiết...).
* **Quản lý lịch phát sóng:** Sắp xếp thứ tự các tin trong một chương trình thời sự, lên lịch ngày giờ phát sóng.
* **Quản lý nhân sự:** Phân quyền cho phóng viên, biên tập viên và quản trị viên.
* **Thống kê & Báo cáo:** Tổng hợp số lượng tin bài, thời lượng phát sóng theo từng chuyên mục.

---

##  Yêu cầu hệ thống (Prerequisites)
Để chạy được mã nguồn này, máy tính cần đáp ứng:
* **Hệ điều hành:** Windows 10/11.
* **Công cụ lập trình:** Visual Studio (Khuyên dùng phiên bản 2019 hoặc 2022).
* **Nền tảng:** .NET Framework hoặc .NET Core/6.0+ (Tùy phiên bản bạn sử dụng).
* **Cơ sở dữ liệu:** SQL Server (Nếu dự án có sử dụng database).

---

##  Hướng dẫn cài đặt (Installation)

Bạn có thể cài đặt và chạy thử nghiệm dự án theo các bước đơn giản sau:

**Bước 1: Tải mã nguồn**
* Nhấn vào nút **Code** (màu xanh lá) phía trên repository và chọn **Download ZIP**, sau đó giải nén file.
* Hoặc clone về máy bằng lệnh Git:
    ```bash
    git clone [Link-Repository-Cua-Ban]
    ```

**Bước 2: Mở dự án**
* Truy cập vào thư mục vừa giải nén.
* Tìm file Solution có đuôi là **`.sln`** 
* Nhấn đúp chuột để mở file này bằng **Visual Studio**.

**Bước 3: Cấu hình và Chạy**
* Nếu có file script SQL, hãy chạy nó trong SQL Server Management Studio (SSMS) để khởi tạo dữ liệu.
* Trong Visual Studio, nhấn phím **F5** hoặc nút **Start** trên thanh công cụ để biên dịch và chạy chương trình.
